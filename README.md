# projeto-pontual-telecom

Criamos uma solução com o objetivo de auxiliar na organização do estoque de equipamentos e utensílios da empresa Pontual Telecom.

A ideia é centralizar todas as movimentações e burocracias da empresa, referentes ao estoque da empresa, numa aplicação (programa) instalada no computador do funcionário responsável.

A aplicação conta com um sistema de identificação, catalogação, status e validação dos equipamentos nele registrados.
